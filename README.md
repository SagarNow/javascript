# javascript
A language that makes websites interactive.it adds behavior (like clicking buttons, showing popups, moving sliders, etc.)
---

- The console.log(): method is used to print messages to the browser’s developer console. 

- console.warn(): The console.warn() method is used to print warning messages to the browser’s developer console. 
- console.error(): The console.error() method is used to print error messages to the browser’s developer console
___
variable: A variable is a container for storing data. 
It can be used to store any type of data, such as numbers, strings, objects, arrays, etc.
___
__let:__ The let keyword is used to declare a variable. It is block-scoped, which means that it is only accessible within the block in which it is declared.cant be redeclared and can be reinitialized.

__const:__ The const keyword is used to declare a constant. It is block-scoped, which means that it is only accessible within the block in which it is declared.cant be redeclared or reinitialized.

__var:__ The var keyword is used to declare a variable. It is function-scoped, which means that it is only accessible within the function in which it is declared.cant be redeclared but can be reinitialized.
___
initialization: The process of assigning a value to a variable is called initialization.
declaration: The process of declaring a variable is called declaration.

ex. declaration: var x;

ex. initialization: var x = 10;
___
### rules for naming variables
1. variables names are case sensitive, "a" and "A" are different variables.
2. variables names must start with a __letter__, an __underscore__ or a __dollar__ sign $.
3. variables only latter, numbers and underscores are allowed. not even space.
4. variables names cannot be reserved keywords.ex. let, const,console, var, function, class, etc.
5. variables names should be descriptive and meaningful.
6. variables names should be unique.
